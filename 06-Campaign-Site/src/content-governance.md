# Campaign citation governance

All quantitative, legal/regulatory, market-size, or “first/leading” claims in campaign copy must include a source.

## Rules

1. Add a citation entry with publication title, publication/source date, and direct URL.
2. Surface that source in UI where claim appears (inline footnote or slide `Sources` panel).
3. Update `/references` in the same pull request whenever claim text or source changes.
4. Remove or rewrite any claim that cannot be verified by a credible source.

## PR checklist

- [ ] New/edited claims have source metadata.
- [ ] Slide-level source links render correctly.
- [ ] `/references` mappings are updated.
