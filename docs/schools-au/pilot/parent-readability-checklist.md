# MetaPet Schools Parent Readability Checklist

Ask one or two parents or carers to read the family-facing materials and complete this checklist.

## Read These First

- Parent Note
- Parent/Carer Privacy Notice
- Family Participation Protocol

## Readability Checks

| Question | Result | Notes |
| --- | --- | --- |
| Can the reviewer explain what the tool is after one read? | Pass / Fail |  |
| Can the reviewer explain what information stays on the device? | Pass / Fail |  |
| Can the reviewer explain who controls any export? | Pass / Fail |  |
| Is the participation process clear? | Pass / Fail |  |
| Is the opt-in or non-participation path understandable? | Pass / Fail |  |
| Is it clear who the family should contact with questions? | Pass / Fail |  |
| Does any sentence sound too technical or too vague? | Pass / Fail |  |

## Ask These Questions

- What part was hardest to understand?
- What question would you still ask the school?
- Does the participation process feel clear and fair?
- Would you know what happens if your child is not the primary device user?
