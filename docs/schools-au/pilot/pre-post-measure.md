# MetaPet Schools Pre/Post Measure

Use the same items before the sequence begins and after it ends.

## Student Confidence Check

Mark one response for each statement.

- `Not yet`
- `Somewhat`
- `Confident`

| Statement | Before | After |
| --- | --- | --- |
| I can explain that a digital system changes when a user gives it input. |  |  |
| I can read a companion state and decide what action makes sense next. |  |  |
| I can describe a simple pattern or feedback loop in a system. |  |  |
| I can name one online safety or digital responsibility habit. |  |  |
| I can suggest a calm or recovery strategy when a system or person looks overloaded. |  |  |

## Teacher Note

If using this with younger students, read each item aloud and allow students to point, mark, or answer verbally.
