# MetaPet Schools Teacher Dry-Run Checklist

Use this before any live pilot. One teacher should complete this rehearsal without heavy coaching.

## Setup

- Open `/schools`
- Find the teacher guide and lesson cards
- Open `/school-game`
- Add aliases only
- Queue one lesson
- Locate the local summary and delete controls

## Timed Rehearsal

| Task | Target | Result | Notes |
| --- | --- | --- | --- |
| Find the school overview | under 1 minute | Pass / Fail |  |
| Find the runtime | under 1 minute | Pass / Fail |  |
| Add aliases and queue a lesson | under 5 minutes | Pass / Fail |  |
| Explain the classroom purpose in plain language | clear on first attempt | Pass / Fail |  |
| Run one lesson flow | 15-20 minutes | Pass / Fail |  |
| Find deletion/reset controls | under 1 minute | Pass / Fail |  |

## What To Watch For

- Any step that feels too technical
- Any confusion between reviewer docs and classroom runtime
- Any extra admin burden
- Any unclear language around privacy, participation, or deletion

## Debrief Prompts

- What took longer than expected?
- Which instruction was least clear?
- What would stop you from using this in a normal school week?
- Could you explain the tool to a parent in one minute?
- Could you tell where to delete local class data without help?
