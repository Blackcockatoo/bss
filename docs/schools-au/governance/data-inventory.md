# MetaPet Schools Data Inventory

| Item | Description | Identifiability | Default location | Purpose |
| --- | --- | --- | --- | --- |
| Learner alias | Teacher-created classroom alias | Low | Local device | Run a class roster without full names |
| Lesson queue | Teacher-selected lessons and timing | Low | Local device | Prepare classroom use |
| Progress state | `not started`, `in progress`, `complete` | Low | Local device | Track pilot delivery |
| Local class summary | Counts and completion totals | Low | Local device | Support pilot evidence |
| Retention metadata | Last local-use timestamp | Low | Local device | Support expiry and deletion |

## Out of scope for default school use

- Student email
- Student password
- Public profile
- Open chat transcript
- Social feed data
