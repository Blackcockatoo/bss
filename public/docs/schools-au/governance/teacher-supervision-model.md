# MetaPet Schools Teacher Supervision Model

## Default supervision expectation

All classroom use is teacher-led. Students should not use `MetaPet Schools` independently, unsupervised, or as an always-on background tool.

## Teacher responsibilities

- Set up aliases only
- Select the lesson and runtime mode
- Explain the classroom purpose in plain language
- Watch for confusion, distress, or misuse
- Review local summaries before any export
- Delete local classroom data when the pilot cycle ends

## Recommended format

- Whole-class demonstration or pairs
- One device per class group where practical
- 20-minute lesson window
- Teacher-controlled start and finish

## Not recommended

- Open free play without a lesson purpose
- Take-home use
- Student-managed account setup
- Public display of individual progress states
