## Cryptographic Verification

All addons have been assigned unique SHA256 cryptographic signatures for verification purposes. These signatures ensure the integrity and authenticity of each addon item.

### Verification Process

To verify an addon's authenticity:

1. Extract the addon ID and name
2. Concatenate them with a pipe separator: `{addon_id}|{name}`
3. Generate the SHA256 hash of this string
4. Compare with the provided SHA256 value

### Example Verification

```
Input: custom-addon-1008|Quantum Entanglement Scarf
SHA256: 6b5d3d5cad2a0e0bf3778685dd696c66faf7064ba9e9566d359a653d6681a1ff
```
